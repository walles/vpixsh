# `vpixsh`

`vpixsh` is a shell featuring:

- Fully `bash` compatible command line syntax (unlike `fish`)
- No scripting support
- Configuration in `~/.vpixsh.yaml`
  - Default environment variables
  - `$PATH` value
- `git` friendly prompt
